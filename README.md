# homebrew-tap

Homebrew tap for tools published by [Yicru](https://github.com/Yicru).

## Install

```bash
brew tap Yicru/tap
brew install Yicru/tap/ghqv
```

Or directly:

```bash
brew install Yicru/tap/ghqv
```

## Available formulae

| Formula | Description |
|---|---|
| [`ghqv`](https://github.com/Yicru/ghqv) | Virtual monorepo workspace manager for ghq-managed repositories |

## Notes

Each formula is generated automatically from its upstream repository's release
workflow. Do not edit formula files by hand.
